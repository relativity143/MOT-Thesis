# MOT Thesis — 磁光阱研究论文整理

本仓库收录了与**磁光阱（Magneto-Optical Trap, MOT）**相关的研究论文与参考资料，用于论文写作与文献调研。

## 内容简介

磁光阱（MOT）是原子物理与量子光学领域的核心实验技术，利用激光冷却与磁场梯度将原子囚禁并冷却至微开尔文量级。本仓库整理了该领域的重要文献，涵盖理论基础、实验实现与前沿应用。
